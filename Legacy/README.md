# StashMaster V2

Gestion modulaire des métadonnées performers (phase 1/phase 2).

- Phase 1 : métadonnées usuelles (Nom, Désambiguïsation, Alias, Date de naissance, Date du décès, Pays, Ethnicité, Couleur des cheveux, Couleur des yeux, Taille, Poids, Mensurations, Faux seins, Durée de carrière)
- Phase 2 : champs avancés (Bio, Trivia, Awards, Tattoos, Piercings, Tags, URLs, Details)

Structure prête pour extensions (group, scene, etc.).
